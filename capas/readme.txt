DATA SOURCE SHAPEFILES:
------------------------
Sample shapefile data taken from CD-ROM 'ESRI Data & Maps'

DCW, ArcWorld Supplement; 
License:
Public domain data from U.S. government are freely 
distributable with proper source attribution or metadata. 

Included data sets:
* country.shp
* cities.shp
* mjrivers.shp

The 'country' and 'mjrivers' shapefiles are slightly
modified (generalized) to decrease file size. 


The data was projected with the following parameters:
* Projection System: Lambert Azimuthal Equal Area
* Central Meridian: 9 degree
* Reference Latitude: 48 degree
* Units: meters
